LeeThemeWood
=============
A wood theme for elgg.

Requirements
=============

* LiangLeeFramework 1.1.6
* LiangLeeThemeDesigner 1.0.2

Installation
============
If you don't have Framework and themedesginer first install them.

* Download LeeThemeWood
* Extract the package in mod directory.
* Enable LeeThemeWood
